# Wikipedia-Golf

ウィキペディアのゴルフができるサイト

[ここから遊べるよ](https://wikipedia-golf.vercel.app/)

## 今日のお題について

「今日のお題」は、毎日午前0時（JST）に自動生成されます。

- GitHub Actionsにより`public/daily-challenge.json`が更新されます
- カテゴリページやユーザーページなどを除外し、通常の記事のみを選択します
- 読み込み時間を短縮し、より楽しいお題を提供します

詳細は[scripts/README.md](scripts/README.md)をご覧ください。
